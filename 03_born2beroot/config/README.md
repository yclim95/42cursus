## 1. Installing Sudo

1.1. Enter Paraphraase

```
Enter your paraphrase
```

1.2. login as user

```
Login user:
Password:
```

![Login as user](https://i.imgur.com/rW3Pa3j.png)


1.3.  Gaining Root Access in the Terminal

Enter ```su -```

```
$ su -
Password:
```

![Root access](https://i.imgur.com/SAJh5QH.png)


1.4. Install Sudo

```
$ apt-get update -y
$ apt-get upgrade -y
$ apt install sudo
```
